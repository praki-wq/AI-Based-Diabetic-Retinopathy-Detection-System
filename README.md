# AI-Based-Diabetic-Retinopathy-Detection-System
An intelligent deep learning-based web application for detecting and classifying Diabetic Retinopathy (DR) from retinal fundus images. This project uses advanced hybrid CNN-Transformer architecture (CoAtNet) trained on the APTOS 2019 Blindness Detection Dataset to identify the severity level of diabetic retinopathy and assist in early diagnosis.
